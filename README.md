# Silent SMS (PWA-ready)

1) Kopieer config:
   cp config.example.json config.json
   # Zet je ANKAREX_TOKEN in config.json

2) Start:
   node server.js
   # Open http://localhost:8080

3) PWA op iOS:
   - Zorg voor HTTPS (of Tailscale) om te installeren als PWA.
   - Safari -> Deel -> Zet op beginscherm.
